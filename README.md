# DHTMLX Richtext with Vue Demo

[![dhtmlx.com](https://img.shields.io/badge/made%20by-DHTMLX-blue)](https://dhtmlx.com/)

## How to start

Clone the repository or download files.

```
yarn
yarn start
```

or

```
npm install
npm run start
```
